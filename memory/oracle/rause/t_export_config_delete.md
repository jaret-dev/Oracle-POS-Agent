# Deleting Export Configurations

[Previous](t_export_config_change_owner.htm) [Next](t_tutorial_add_export_config.htm) JavaScript must be enabled to correctly display this content

You must be the owner of the export configuration to delete it and the configuration cannot be active.

1.  In Reporting and Analytics, click the side navigation menu, click Reports, click Exports, click Configurations, and then click the configuration in the list.
2.  Click Delete and then click Delete.

**Parent topic:** [Export Data from Reporting and Analytics](c_export_data.htm "Set up the data export process and run exports to an endpoint to manage data.")
