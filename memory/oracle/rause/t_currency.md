# Adding a Currency Type

[Previous](t_currency_base.htm) [Next](t_currency_exchangerate.htm) JavaScript must be enabled to correctly display this content

1.  Make sure you enabled common currency.
2.  In Reporting and Analytics, click the side navigation menu, click Configuration, click Warehouse Admin, click Miscellaneous, and then click Currency.
3.  Fill out the form as described in the following table:
    
    Field
    
    Description
    
    Currency Name
    
    Enter the name of the currency type. For example, American Dollar.
    
    Abbreviation
    
    Optionally, enter an abbreviation for the currency. For example, USD.
    
    Multiply
    
    Select to multiply the currency against the base currency.
    
    Round To
    
    Enter the number of decimal places for rounding in the 0.0 format. For example, 0.00 to round to two decimal places and 0.0 to round to one decimal place.
