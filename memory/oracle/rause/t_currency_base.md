# Enabling Currency Configuration and Setting the Common Currency

[Previous](c_currency.htm) [Next](t_currency.htm) JavaScript must be enabled to correctly display this content

1.  In Reporting and Analytics, click the side navigation menu, click Configuration, click Warehouse Admin, click Miscellaneous, and then click Enterprise Options.
2.  Select Enable Common Currency.
3.  Select the currency type from the Common Currency drop-down list.
    
    This is the currency to which other currencies are exchanged and converted.
