# Scheduled Reports

[Previous](t_data_privacy_logs.htm) [Next](t_report_mail.htm) JavaScript must be enabled to correctly display this content

You can use Report Mail to schedule reports to run at a specified date and time, and to distribute the report by email to a user or role.

For example, you can configure Report Mail to generate and send a report every morning that summarizes operations from the previous business day. This lets you access reports on a regular basis without logging into Reporting and Analytics or manually running the report.

Note:

Chart visualizations are not included in Report Mail.

-   [Scheduling Reports for Email Distribution](t_report_mail.htm)  
    Set up report mail and define how often the emails are sent.
