# Navigating a List

[Previous](c_accessibility_multiple_selections.htm) [Next](c_accessibility_check_box.htm) JavaScript must be enabled to correctly display this content

-   Use the F2 key to navigate into a list.
    
-   Use the tab key to navigate within the list.
    
-   Use the Esc key to navigate out of the list.
