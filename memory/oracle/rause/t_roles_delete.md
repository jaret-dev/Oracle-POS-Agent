# Delete Roles

[Previous](t_roles_edit.htm) [Next](c_user_mgmt.htm) JavaScript must be enabled to correctly display this content

Required system privileges: View/Add/Edit/Delete Roles.

1.  In Reporting and Analytics, click the side navigation menu, click Administration, and then click Roles.
2.  Select the role you want to delete.
3.  Confirm that there are no users assigned to the role.
4.  Click Delete and then click Delete.

**Parent topic:** [Control Who Sees What](c_access_controls.htm "Define the privileges and access users have within each role.")
