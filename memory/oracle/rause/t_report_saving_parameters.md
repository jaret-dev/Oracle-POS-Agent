# Saving Report Parameters

[Previous](t_reports_running.htm) [Next](t_reports_exporting.htm) JavaScript must be enabled to correctly display this content

1.  In Reporting and Analytics, click the side navigation menu, click Reports, click My Reports or Library, and then select the report.
2.  Click to the right of Parameter Name and select one of the following:
    1.  Save — to save these parameters and run the report with them in the future.
    2.  Save As — to create a new version of this report that runs with your preferred parameters. After clicking Save As, enter a name for the parameter that will appear on the report tile, and click Save and Run. Your new version will appear in My Reports and in the Library with a saved icon ![This is an image of the saved icon.](img/saved_icon.png "This is an image of the saved icon."). The original version will also appear.
    3.  Restore Default — to return the parameters to their original settings.
    4.  Delete Save As — to delete the Save As version.
