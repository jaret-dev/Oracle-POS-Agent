# Set the Functional Administrator

[Previous](t_set_hierarchy_admin.htm) [Next](t_roles_compare.htm) JavaScript must be enabled to correctly display this content

Required system privilege: View/Add/Edit/Delete Roles.

Can only be assigned by a current Functional Administrator.

1.  In Reporting and Analytics, click the side navigation menu, click Administration, and then click Roles.
2.  Click the role you want to set as a Functional Administrator.
3.  Select Functional administrator and then click Save and Close or Save.

**Parent topic:** [Control Who Sees What](c_access_controls.htm "Define the privileges and access users have within each role.")
