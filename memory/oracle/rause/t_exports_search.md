# Searching for Exports

[Previous](t_export_on_demand.htm) [Next](t_export_config_change_owner.htm) JavaScript must be enabled to correctly display this content

Search for on-demand exports or configurations run with schedules.

1.  In Reporting and Analytics, click the side navigation menu, click Reports, and then click Exports.
2.  To search for on-demand exports, click On Demand and then enter search criteria.
3.  To search for configurations run with a schedule, click Search and then enter search criteria.

**Parent topic:** [Export Data from Reporting and Analytics](c_export_data.htm "Set up the data export process and run exports to an endpoint to manage data.")
