# Deactivate or Activate a User

[Previous](t_user_management_edit.htm) [Next](c_password_requirements.htm) JavaScript must be enabled to correctly display this content

The status of a user is active or inactive. For example, a seasonal worker can have an active status while working in the summer and have an inactive status for rest of the year.

Required system privilege: View/Add/Edit Users.

1.  In Reporting and Analytics, click the side navigation menu, click Administration, and then click Users.
2.  Search for the user and select the user.
3.  Click Inactive to deactivate the user or click Active to reinstate the user.
4.  Click Save and Close or Save.
