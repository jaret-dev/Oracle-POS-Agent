# Running Reports

[Previous](t_reports_build_transfer.htm) [Next](t_report_saving_parameters.htm) JavaScript must be enabled to correctly display this content

You can enable auto run if you want the report to use the same parameters, like location or business date, each time someone opens the report. Disable auto run if you expect most users to change the parameters when they view the report. For example, an organization with hundreds of locations may want to disable auto run so a user does not have to wait for the report to run before editing the parameters and running it again for their specific location.

1.  In Reporting and Analytics, click the side navigation menu, click Reports, click My Reports or Library, and then select the report.
2.  Click Edit Parameters, select parameters, and then click Run.
3.  To enable auto run, select the auto run icon ![This is an image of the selected auto run icon](img/auto_run_icon.png "This is an image of the selected auto run icon").
4.  To disable auto run, deselect the auto run icon ![This is an image of the deselected auto run icon](img/disable_auto_run.png "This is an image of the deselected auto run icon")
