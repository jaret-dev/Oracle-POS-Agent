# Searching for Guest Checks

[Previous](t_iquery_building.htm) [Next](c_org_hierarchy.htm) JavaScript must be enabled to correctly display this content

Reporting and Analytics lets you retrieve guest check details.

This feature lets you search for guest checks that match certain search criteria. For example, if a customer calls to get a copy of their receipt you can narrow down the list of guest checks based on search criteria.

1.  In Reporting and Analytics, click the side navigation menu, click Reports, and then click Guest Checks.
2.  Specify the search criteria and then click Search.
3.  Select the check to view the Check Detail report.
    
    Within the report, you can download or print the report. If using Simphony, you can drill down further to see the POS transaction view. For more details about the Check Detail report, see the Oracle MICROS Reporting and Analytics Reports Reference Guide.
