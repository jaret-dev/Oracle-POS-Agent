# Adding and Assigning Dashboards

[Previous](c_dashboards.htm) [Next](t_dashboards_customize.htm) JavaScript must be enabled to correctly display this content

You can add up to 12 report tiles to automatically populate the dashboard for each role. Users can then manage their own dashboard and select different tiles later.

1.  In Reporting and Analytics, click the side navigation menu, click Administration, click Roles, and then click the role.
2.  Click Reports Dashboard.
3.  Search for report tiles and click Select to add tiles to the dashboard for this role.
4.  Click Save and Close or Save.
