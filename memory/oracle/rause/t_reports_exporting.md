# Exporting Reports

[Previous](t_report_saving_parameters.htm) [Next](c_report_msexcel.htm) JavaScript must be enabled to correctly display this content

You can export reports into Microsoft Excel, save them as a PDF, or print them.

1.  In Reporting and Analytics, click the side navigation menu, click Reports, click My Reports or Library, and then select the report.
2.  To download the report, click the ![This is an image of the download icon.](img/download_icon.png "This is an image of the download icon.") icon.
    1.  Select Microsoft Excel (.xlsx) and then open the downloaded file to export the data as it appears in the report.
    2.  Select Comma-Separated Values (csv) and then open the downloaded file to export the data in plain text.
3.  To save as a PDF or print the report, click the ![This is an image of the printer icon.](img/printer_icon.png "This is an image of the printer icon.") icon.
