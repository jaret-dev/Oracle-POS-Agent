# Edit Users

[Previous](t_user_mgmt_view.htm) [Next](t_user_management_deactivate.htm) JavaScript must be enabled to correctly display this content

Required system privilege: View/Add/Edit Users.

1.  In Reporting and Analytics, click the side navigation menu, click Administration, and then click Users.
2.  Search for the user and select the user.
3.  Edit the user information and click Save and Close or Save.
