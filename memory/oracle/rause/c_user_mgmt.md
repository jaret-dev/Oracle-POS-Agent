# User Management

[Previous](t_roles_delete.htm) [Next](t_user_mgmt_add.htm) JavaScript must be enabled to correctly display this content

Add users, assign users to a role, edit and manage users.

In Reporting and Analytics, each user is uniquely identified with a user name and password. The user name and password, combined with a role, determine what tasks the user can perform and what data the user can see.

As a Reporting and Analytics administrator, you can manage users in your enterprise and do the following:

-   [Add Users](t_user_mgmt_add.htm)  
    
-   [View Users](t_user_mgmt_view.htm)  
    
-   [Edit Users](t_user_management_edit.htm)  
    
-   [Deactivate or Activate a User](t_user_management_deactivate.htm)  
    
-   [Passwords and Locked Accounts](c_password_requirements.htm)  
    Lists the criteria for passwords and includes locked account details.
