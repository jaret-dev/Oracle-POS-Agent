# Data Privacy

[Previous](t_currency_exchangerate.htm) [Next](t_data_privacy_deidentify.htm) JavaScript must be enabled to correctly display this content

De-identify users and download personal data reports.

Reporting and Analytics stores personally identifiable information such as names, email addresses, and sign-in date and time stamps. Reporting and Analytics provides tools that let you quickly address user privacy requests. For example, you can de-identity users. When you de-identify a user, the system overwrites their information with generic or empty values so activity cannot be traced back to the user.
