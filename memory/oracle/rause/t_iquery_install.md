# Installing iQuery

[Previous](c_iquery_permissions.htm) [Next](t_iquery_building.htm) JavaScript must be enabled to correctly display this content

1.  Close Microsoft Excel if it is open.
2.  In Reporting and Analytics, click the side navigation menu, click Reports, and then click either My Reports or Builder.
3.  Click Download iQuery and then click Download.
4.  Double-click the `iQuerySetup.exe` file and then follow the prompts to install the software.
5.  Validate the installation:
    1.  Open Microsoft Excel.
    2.  Locate the iQuery plug-in, and then open iQuery.
        
        The location varies depending on the version of Microsoft Excel. For example, in Microsoft Excel 2013, access iQuery from the ADD-INS ribbon.
        
    3.  Log in to iQuery.

**Parent topic:** [Custom Reports in Microsoft Excel](c_report_msexcel.htm "iQuery is a Microsoft Excel add-in that lets you export data to Microsoft Excel and then create reports using selected data subject areas.")
