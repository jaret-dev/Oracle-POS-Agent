# Reset Another User’s Password

[Previous](c_password_requirements.htm) [Next](t_user_management_password_reset_yours.htm) JavaScript must be enabled to correctly display this content

Required system privilege: View/Add/Edit Users.

1.  In Reporting and Analytics, click the side navigation menu, click Administration, and then click Users.
2.  Search for the user and select the user.
3.  Click Reset Password and then click Reset Password.
