# Set the Hierarchy Administrator

[Previous](c_roles_set_hierarchy_functional_admin.htm) [Next](t_set_functional_admin.htm) JavaScript must be enabled to correctly display this content

Must be the current Hierarchy Administrator to reassign.

Required system privilege: View/Add/Edit/Delete Roles.

1.  In Reporting and Analytics, click the side navigation menu, click Administration, click Roles, and then click Assign the Hierarchy Administrator.
2.  Select the role you want to assign as the Hierarchy Administrator and then click Save and Close.

**Parent topic:** [Control Who Sees What](c_access_controls.htm "Define the privileges and access users have within each role.")
