# Creating Category Groups

[Previous](t_report_hierarchy_creating.htm) [Next](t_categorygroups_hierarchy.htm) JavaScript must be enabled to correctly display this content

1.  In Reporting and Analytics, click the side navigation menu, click Configuration, click Warehouse Admin, click Category Groups, and then click Category Groups.
2.  Click Add.
3.  Enter a name for the category group and then click Save.
