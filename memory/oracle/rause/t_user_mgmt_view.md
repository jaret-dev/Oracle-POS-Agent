# View Users

[Previous](t_user_mgmt_add.htm) [Next](t_user_management_edit.htm) JavaScript must be enabled to correctly display this content

Required system privilege: View/Add/Edit Users.

1.  In Reporting and Analytics, click the side navigation menu, click Administration, and then click Users.
2.  Type or select search criteria and then click Search to filter the search results.
    
    For example, to view inactive users for an enterprise role named Enterprise Manager, select Enterprise Manager in the Role field and then select Inactive.
    
    To search for a specific user, enter their name or system user name and then click Search.
