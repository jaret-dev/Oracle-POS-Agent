# Using the Tab Key

[Previous](c_accessibility.htm) [Next](c_accessibility_switch_tabs.htm) JavaScript must be enabled to correctly display this content

Use the Tab key to navigate through Reporting and Analytics screens. The default tab order flows from left to right and top to bottom.
