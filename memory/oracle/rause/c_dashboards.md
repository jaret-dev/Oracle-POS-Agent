# Dashboards

[Previous](c_reports.htm) [Next](t_dashboards_add.htm) JavaScript must be enabled to correctly display this content

Dashboards contain tiles that show snippets of data from reports. Your system administrator configures a default set of tiles to display on the dashboard according to your role. You can customize the dashboard by adding and removing tiles or by changing the order in which the tiles appear on the dashboard. If you need more information for a tile, click the link under the tile title to view the report that supplies the data to the tile.
