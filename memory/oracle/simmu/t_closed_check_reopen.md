# Reopening a Closed Check

[Previous](t_closed_check_adjust.htm) [Next](t_closed_check_reprint.htm) JavaScript must be enabled to correctly display this content

1.  To reopen a check using the check number, press the Reopen Closed Check function key, enter the closed check number, and then press OK.
2.  To view a list of closed checks, press the Reopen Closed Check From List function key, select the check to reopen, and then press OK.
3.  Make the necessary changes to the check, and then close the check.
