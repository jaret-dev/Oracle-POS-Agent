# Auditing a Till

[Previous](t_ecm_adjust_starting_amount.htm) [Next](t_ecm_pull_cash.htm) JavaScript must be enabled to correctly display this content

You can perform a mid-shift audit or count of a till without closing the till, even when the Close on Count Completion option is configured in the Templates module of the EMC.

1.  Select the function for till report, and then click Print.
2.  Select the function to audit a receptacle (till).
3.  Enter a reference for the count, and then click OK.
4.  Enter the values of each unit, and then click Done.
5.  Select the function for till report, and then click Print. The till balance and count are updated on the Till Report, and the till session remains open.
