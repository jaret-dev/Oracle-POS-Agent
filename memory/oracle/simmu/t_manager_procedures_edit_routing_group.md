# Editing Routing Groups

[Previous](t_manager_procedures_redirect_order_device.htm) [Next](t_manager_procedures_adjust_mi_definitions.htm) JavaScript must be enabled to correctly display this content

1.  Press the Edit Routing Groups function key, or press the PMC Launch function key, and then press the Edit Routing Groups button on the Functions tab.
2.  Select the workstation and the routing group.
3.  Select the new order device, and then press Save.
