# Redirecting Orders

[Previous](t_manager_procedures_replay_checks.htm) [Next](t_manager_procedures_edit_routing_group.htm) JavaScript must be enabled to correctly display this content

1.  Press the Redirect Order Device function key, or press the PMC Launch function key, and then press the Edit Redirect Order Device button on the Functions tab.
2.  Select the revenue center, select an order device from the list, and then press Edit.
3.  Select the device to which you want to redirect the orders that come to the device you selected in Step 2, and then press OK.
