# Adjusting the Menu Item Price on a Check

[Previous](t_manager_procedures_adjust_mi_availability.htm) [Next](t_manager_procedures_barcodes.htm) JavaScript must be enabled to correctly display this content

1.  Select a menu item from the check detail area, and then press the Menu Item Price Override function key.
2.  Enter the new price, and then press OK.
3.  (Optional) Enter reference information, and then press OK.
