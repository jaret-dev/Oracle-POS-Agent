# Viewing ECM Alerts and Summary with the Engagement Cloud Service

[Previous](c_ecm_no_sale_operation.htm) [Next](c_ecm_swap_tablets_with_base_station.htm) JavaScript must be enabled to correctly display this content

If the property has configured the Engagement Cloud Service, you can view Enterprise Cash Management (ECM) alerts and summary information for till and server bank receptacles through Engagement.

1.  Open Engagement or sign on to Engagement using a tablet device.
2.  On the Hub page, click the ECM alert widget or the ECM summary widget. The following table describes the ECM alert and summary widgets.
    
    Table 4-2 ECM Hub Page Widgets
    
    ECM Widgets for Hub Pages
    
    Description
    
    ECM Alert
    
      
    ![This figure shows the Enterprise Cash Management alerts icon.](img/ecm-alert.png "This figure shows the Enterprise Cash Management alerts icon.")  
    
    The ECM alert widget allows you to view ECM alerts based on cash pull thresholds for till and server bank receptacles. This widget is supported for multiple cash management classes. The ECM alert widget shows:
    
    -   Receptacle name
        
    -   Alert type
        
    -   Alert level (if applicable)
        
    -   Variance (threshold value – actual receptacle session amount)
        
    
    The ECM alert widget also indicates when there are no active alerts.
    
    ECM Summary
    
      
    ![This figure shows the Enterprise Cash Management summary icon indicating active and offline sessions.](img/ecm-summary.png "This figure shows the Enterprise Cash Management summary icon indicating active and offline sessions.")  
    
    The ECM summary widget allows you to view summary information for active ECM receptacles. This widget is supported for multiple cash management classes. The ECM summary widget shows:
    
    -   Receptacle name
        
    -   Number of open receptacle sessions for each receptacle type
        
    -   Current total in base class amount
        
    
    The ECM summary widget also indicates:
    
    -   Active alerts for a session
        
    -   Offline open receptacle sessions
        
    -   Multiple classes
