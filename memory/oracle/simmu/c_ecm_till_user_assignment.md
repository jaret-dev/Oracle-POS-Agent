# Till and User Assignment

[Previous](t_ecm_adjust_reference.htm) [Next](t_ecm_assign_till_cash_drawer.htm) JavaScript must be enabled to correctly display this content

You can assign a till to a cash drawer and assign users to a till using several methods:

-   Manually
    
-   Using Quick Start
    
-   Using Allow All
    

You must assign the till to a cash drawer and assign a user to the till in order to perform cash transactions on the till.
