# No Sale Operation

[Previous](t_ecm_view_receptacle_status.htm) [Next](t_ecm_engagement_alerts_summary_widgets.htm) JavaScript must be enabled to correctly display this content

The No Sale operation allows a privileged workstation operator to open the cash drawer outside of a transaction without performing other operations. The workstation operator typically uses this function key to make change for a customer. When configured, the workstation operator must enter a No Sale reason to open the cash drawer.

No Sale operations are recorded for:

-   The workstation operator’s assigned receptacle
    
-   Reporting purposes
