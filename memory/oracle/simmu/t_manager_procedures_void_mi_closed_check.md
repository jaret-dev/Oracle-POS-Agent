# Voiding All Menu Items From a Closed Check

[Previous](t_manager_procedures_transfer_android_logs_to_caps.htm) [Next](t_manager_procedures_reprint_time_cards.htm) JavaScript must be enabled to correctly display this content

1.  If you know the check number:
    1.  Press the Void Check function key, enter the check number, and then press OK.
    2.  Press OK again, and then press Yes to confirm.
2.  If you do not know the check number:
    1.  Press the Void Check by List function key, select the check from the list, and then press OK.
    2.  Press Yes to confirm.
