# Replaying Checks

[Previous](t_manager_procedures_support_tab.htm) [Next](t_manager_procedures_redirect_order_device.htm) JavaScript must be enabled to correctly display this content

1.  Press the Launch PMC function key.
2.  On the Functions tab, press Replay Checks.
3.  Select one of the following locations:
    
    -   Workstation: Replays checks stored in the workstations
        
    -   Check and Posting Service: Replays checks stored in the Check and Posting Service (CAPS) workstation
        
    -   Check and Posting Backup Service: Replays checks stored in the CAPS backup workstation
        
    
4.  Select the workstation, the date, and the time period for which you want to replay the checks.
5.  To find the current check replay status for the period that you selected in Step 4, press View Transaction State.
6.  Press Replay Checks, and then press OK to confirm.
