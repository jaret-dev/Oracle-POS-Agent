# Running Start of Day Manually

[Previous](t_cruise_update_reload_workstation_database.htm) [Next](c_closed_check_operations.htm) JavaScript must be enabled to correctly display this content

You can run the Start of Day manually from the Enterprise level of the EMC or locally from the workstation.

1.  To manually run the Start of Day from the Enterprise level:
    1.  From the EMC, select the Enterprise level, click Tasks, and then click Run PC Autosequences.
    2.  Select Increment Business Day from the Choose PC Autosequence drop-down list, and then click Run.
2.  To manually run the Start of Day locally on a workstation:
    1.  Sign on to the workstation at the appointed time to increment SOD.
    2.  Press the Start of Day function key.
    3.  Press Yes at the prompt to confirm.
