# Tablet Swaps with the Oracle MICROS Base Station

[Previous](t_ecm_engagement_alerts_summary_widgets.htm) [Next](c_ecm_offline_cm_operations.htm) JavaScript must be enabled to correctly display this content

You can use ECM on the Oracle MICROS Tablet E-Series 11-inch and Oracle MICROS Base Station. The workstation operator can dock and undock multiple E-Series tablets from the Base Station to allow mobile activities. The till session assigned to a cash drawer persists while swapping tablets on a single Base Station.
