# Performing a Manual End of Day (EOD) Settlement

[Previous](t_manager_tasks_change_kitchen_theme.htm) [Next](t_manager_tasks_enable_disable_srm.htm) JavaScript must be enabled to correctly display this content

If you are using the Oracle Payment Interface (OPI) and EOD credit card settlement fails, you can a manually perform the settlement.

1.  Sign on to a workstation.
2.  Click the EOD Settlement function key.
