# Adjusting a Count Sheet

[Previous](t_ecm_adjust_count.htm) [Next](t_ecm_deposit_cash.htm) JavaScript must be enabled to correctly display this content

You can change the count sheet that is used to perform a count.

1.  Prior to counting, select the function to adjust a count sheet.
2.  If multiple receptacles are available, highlight the appropriate receptacle, and then click OK.
3.  Select the count sheet you want to use, and then click OK.
