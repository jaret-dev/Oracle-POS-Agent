# Reprinting a Closed Check

[Previous](t_closed_check_reopen.htm) [Next](c_ecm_operations.htm) JavaScript must be enabled to correctly display this content

You can reprint the check for a closed transaction in the current or previous business dates, and you can reprint a receipt for the last closed check. You need to have the appropriate privileges to perform these operations.

1.  To reprint the check for a closed transaction:
    1.  Press the Reprint Closed Check function key.
    2.  Select the check to print, and then press OK.
2.  To reprint a receipt for the last closed check, press the Reprint Previous Closed Check function key. The receipt prints.
