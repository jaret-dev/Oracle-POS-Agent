# Till Assignment Status with Smart Key

[Previous](t_ecm_allow_access.htm) [Next](t_ecm_adjust_starting_amount.htm) JavaScript must be enabled to correctly display this content

When you implement a smart key, the key determines whether the entire workstation is usable based on the till assignment.

-   When a till is unassigned, the application prompts the workstation operator to assign a till to the cash drawer.
    
-   When a till is assigned, the application prompts the workstation operator to sign on to the workstation.
