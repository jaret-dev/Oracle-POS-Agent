# Creating, Editing, and Assigning Barcodes to Menu Items

[Previous](t_manager_procedures_adjust_mi_price.htm) [Next](t_manager_procedures_change_serving_period.htm) JavaScript must be enabled to correctly display this content

1.  Press the Edit Barcode function key, and then select a revenue center.
2.  Scan or enter a barcode number, and then press Find.
3.  To create a new barcode number, press Yes when prompted.
4.  Press the Select button adjacent to the Menu Item field, select a menu item from the list, and then press Select. You can use the filters to streamline the list.
5.  (Optional) Select a new Price Number.
6.  (Optional) Enter the Alternate Price and the Alternate Prep Cost.
7.  To change the barcode number, press Assign Barcode, scan or enter a new barcode, and then press OK.
8.  Press Save, and then press Done to exit the Edit Barcode screen.
