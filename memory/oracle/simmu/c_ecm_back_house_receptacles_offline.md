# Back of House Receptacles Offline

[Previous](c_ecm_offline_cm_operations.htm) [Next](c_ecm_till_offline.htm) JavaScript must be enabled to correctly display this content

When a workstation is offline from the Cash Management Service, workstation operators cannot perform any functions against a safe, bank deposit, or petty cash receptacle. Operators can perform these functions only from the primary posting workstation that runs the Cash Management Service. The POS application must be accessible from the primary posting workstation in order to perform these operations.
