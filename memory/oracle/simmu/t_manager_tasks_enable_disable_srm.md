# Enabling and Disabling Sales Recording Modules

[Previous](t_manager_tasks_perform_manual_eod_settlement.htm) [Next](c_cruise_lock_unlock_foh_property.htm) JavaScript must be enabled to correctly display this content

You must have the appropriate privileges to perform this task.

1.  Press the Quebec SRM Control function key.
2.  To disable the SRM, select the SRM connected printers, and then press OK.
3.  To enable the SRM, select the SRM connected printers that are disabled, and then press OK.
