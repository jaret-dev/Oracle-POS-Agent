# Employee Management

[Previous](c_preface.htm) [Next](c_manager_tasks_ws_reports_procedures.htm) JavaScript must be enabled to correctly display this content

If you are using

-   Simphony 19.6 or later with Reporting and Analytics 20.1.15 or later
    

and your EMC employees are managed through People Management, use People Management in Simphony Home to add and edit employees. The Oracle MICROS Reporting and Analytics User Guide, located on the [Oracle Help Center](https://docs.oracle.com/en/industries/food-beverage/back-office/20.1/rause/index.html), contains more information about People Management.

If you are using one of the following versions of software

-   Simphony 19.5 or earlier
    
-   Simphony 19.6 or later with Reporting and Analytics 20.1.14 or earlier
    
-   Simphony 19.6 or later with Reporting and Analytics 20.1.15 or later
    

without managing your EMC employees through People Management, you can change the role for an existing employee using the Simphony EMC. [Changing Employee Information](https://docs.oracle.com/en/industries/food-beverage/simphony/19.6/simcg/t_employee_change_info.htm) in the Oracle Simphony Configuration Guide contains more information on changing employee roles.

If you are using Oracle MICROS Labor Management with Simphony 19.5 or earlier, you need to add employees using Labor Management. The Oracle MICROS Labor Management User Guide, located on the [Oracle Help Center](https://docs.oracle.com/en/industries/food-beverage/back-office/20.1/lamug/t_employee_adding.htm), contains information about adding employees in Labor Management. The initial employee information is sent from Labor Management to Simphony.
