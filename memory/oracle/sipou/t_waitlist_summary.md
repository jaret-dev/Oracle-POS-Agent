# Setting the Wait List Summary

[Previous](t_waitlist_abandon.htm) [Next](r_waitlist_icons.htm) JavaScript must be enabled to correctly display this content

## Setting the Wait List Summary

The summary area must be configured in order to view it in the Front of House.

1.  Double-click (or tap) quickly on the Wait List to open it.
2.  Highlight a request in the list. The summary area appears beneath the Wait List and contains the following information about a request.
    
    Table 10-11 Summary Area Options
    
    Summary Option
    
    Shows
    
    Abandoned Time
    
    If the guest leaves before being seated, this summary option shows the date and time the request was abandoned.
    
    Confirmation Number
    
    The confirmation number associated with a reservation request.
    
    Covers
    
    The number of people to be seated for the request.
    
    Creation Date
    
    The date and time the request was created.
    
    Estimated Wait Time
    
    The estimated amount of time remaining until the application anticipates the request being seated.
    
    Greeted Date
    
    The time the request was greeted by a hosting staff member.
    
    Pager Number
    
    The pager number assigned to the guest when hosting staff greeted them.
    
    Phone Number
    
    The guest’s phone number.
    
    Referral Notes
    
    Text included in the referral notes field of the request.
    
    Request Method
    
    A description of the request method (for example, phone ahead, Internet).
    
    VIP
    
    A VIP flag if the guest has VIP status.
    
    Wait Quote
    
    The amount of time the hosting staff quoted to the guest upon arrival, or the amount of time a reservation had from the time they were greeted to the time they are scheduled to be seated.
    
3.  To hide the summary area and prevent it from showing, long-press on a request, and then select Hide Summary Area.
4.  To show the summary area, long-press on a request, and then select Show Summary.
5.  To modify the summary area, long-press on a request, and then select Modify Summary Area.
    1.  Select the appropriate summary options in the Select Items column.
    2.  To show the summary on a separate line in the summary area, select the corresponding Add New Line check box. The text wraps when you deselect Add New Line.
    3.  Click OK.
