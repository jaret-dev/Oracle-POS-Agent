# Using OPERA Self Pay to Close a Check

## Using OPERA Self Pay to Close a Check

The following steps explain the process in which a guest can close a check or room charge after ordering using OPERA Self Pay.

1.  After clicking the button that corresponds to OPERA Self Pay on the ordering screen, the guest can do the following:
    1.  Review the check.
    2.  Select a desired language.
    3.  Click Exit to return to the ordering screen.
2.  Click Next to proceed.
3.  If the Enable Tip Screen option is selected, the guest is prompted to enter a tip. If the Enter Default Tip Percentage option is deselected and a default tip percentage specified, the By Percent field is populated with the default tip percentage. These fields are editable, and are reflected in the Total Due field.
4.  Click Check to review the check after the tip is added.
5.  After reviewing the check, click Next to proceed.
6.  The guest is prompted to enter an email address to receive the optional receipt.
    
    If the Hide Email Prompt Screen option is enabled in Simphony, the guest is not prompted to enter an e-mail address.
    
7.  Click Next.
8.  On the subsequent page, the guests enter the correct room number and last name. If one or both fields is either left empty or has the incorrect guest information, an error message appears.
9.  Click Next. The signature prompt screen appears.
10.  Guests can sign for their room charge using their finger. They may click the Retry button if the signature is illegible or if they want to rewrite their signature.
11.  Click Next. A copy of the receipt with the guest signature is sent through email to the guest if they provided an email address.
12.  After the transaction is processed, the guest is shown the final screen, which confirms a successful payment and shows the check details.
13.  Click Exit.
     1.  If the Exit OPERA Self Pay Without ID Prompt option was enabled, clicking Exit takes the guest back to the order screen.
     2.  If the Exit OPERA Self Pay Without ID Prompt option was disabled, clicking Exit prompts for employee authorization.
