# Menu Item Information Check Detail Selection Plug-in

[Previous](t_menu_items_age_verification.htm) [Next](t_menu_items_toggle_plugin.htm) JavaScript must be enabled to correctly display this content

## Menu Item Information Check Detail Selection Plug-in

If the Menu Item Information extension application plug-in is configured, workstation operators can view and print nutritional and allergen information for menu items already added to the check.

1.  Select a menu item in the check detail to which you want to add the menu item information.
2.  Click Menu Item Information. The menu item information page appears, including the allergen and nutritional information for the menu item (if configured).
    
    The allergen and nutritional information shown is for the selected menu item only. Condiments and modifiers do not augment the allergen and nutrition information.
    
    Figure 4-3 Example of Menu Item Information
    
      
    ![This image shows the menu item information on the workstation.](img/ext_app_menu_allergen.png "This image shows the menu item information on the workstation.")  
    
3.  To add this menu item to the check, click Add.
4.  To print a copy of the information on the receipt printer, click Print.
    
    Figure 4-4 Example of Printed Receipt with Menu Item Information
    
      
    ![This image shows an example of a printed receipt with menu item information.](img/ext_app_receipt_menu.png "This image shows an example of a printed receipt with menu item information.")  
    
5.  To close the menu item information display, click Back.
