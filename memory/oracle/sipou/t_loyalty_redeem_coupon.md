# Redeeming a Coupon

[Previous](t_loyalty_issue_coupons.htm) [Next](t_loyalty_transfer_balance.htm) JavaScript must be enabled to correctly display this content

## Redeeming a Coupon

1.  Begin a check and add items.
2.  Navigate to the gift and loyalty screen, and then select the function key to redeem a coupon.
3.  Enter the coupon number obtained from the Gift and Loyalty Advanced website or email.
