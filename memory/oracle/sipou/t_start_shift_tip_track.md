# Using Tip Track

[Previous](t_start_shift_increment.htm) [Next](t_start_shift_run_reports.htm) JavaScript must be enabled to correctly display this content

## Using Tip Track

Servers can use Tip Track to:

-   Share tips with other employees
    
-   Edit tip-outs
    
-   Claim tips from other employees
    

You cannot use Tip Track with Labor Management Cloud Service.

1.  Click the Tip Track Tip Out function key.
2.  Enter the tip out amount, and then click OK.
3.  From the Select Employee dialog, select the employee to tip-out, and then click OK.
    
    Employees shown in the Select Employee dialog have the Claim Tips From Other Employee privilege.
    
4.  To share the tip with another employee, click Yes at the prompt, and then repeat Steps 2 through 4.
5.  To edit tip-outs given by the signed-on employee:
    1.  Click the Tip Track Edit My Tips function key.
    2.  From the Select The Tip To Adjust dialog, select a tip transaction.
        
        To revise the tip amount, click Edit Tip Amount, enter the new tip amount, and then click OK.
        
        To change the receiving employee, click Edit Employee, select a new receiving employee from the list, click OK, and then click Yes.
        
    3.  Click Done.
6.  To edit the tip-outs given by other employees:
    1.  Click the Tip Track Edit function key.
    2.  In the Select Employee dialog, select the employee that shared the tip.
    3.  In the Select The Tip To Adjust dialog, select a tip transaction.
        
        To revise the tip amount, click Edit Tip Amount, enter the new tip amount, and then click OK.
        
        To change the receiving employee, click Edit Employee, select a new receiving employee from the list, click OK, and then click Yes.
        
    4.  Click Done.
