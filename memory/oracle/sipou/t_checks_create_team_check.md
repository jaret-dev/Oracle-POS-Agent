# Creating a Team Check

[Previous](t_checks_delete_service_team.htm) [Next](t_checks_add_service_team_to_check.htm) JavaScript must be enabled to correctly display this content

## Creating a Team Check

1.  Press the Begin Team Check function key, and then press Yes when prompted for confirmation.
2.  If you belong to multiple service teams, from the Team Service dialog, select the team for which this check belongs, and then press Select.
3.  If you have the privilege to create service teams, click Add to create a new team.
