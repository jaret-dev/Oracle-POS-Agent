## Beginning and Closing a Check

You start a transaction using a function key. You must have the privileges to begin a guest check and to begin a check by table.

1.  To begin a check, sign on to the workstation, and then select one of the function keys described in the following table:
    
    Table 3-1 Begin Check Function Keys
    
    Function Key
    
    Begins a Check ...
    
    400 - Begin Check by Number
    
    By check number. Typically, the workstation automatically generates check numbers in numeric order. If the revenue center has the option Operator Assigns Check Numbers set, you must manually enter the check number.
    
    401 - Begin Check by Table
    
    By table number or table ID. Typically, severs who are assigned to a section of tables use this function to begin a check.
    
    435 - Begin Check by ID
    
    By guest check ID. Typically, bartenders use this function to identify a guest. For example, a bartender can enter an ID to show a guest’s name or shirt color to more easily identify the check on the Open Check Screen Look Up (SLU).
    
2.  To close the check, select the function key to pay.
3.  Tender the check using the guest’s payment method.
