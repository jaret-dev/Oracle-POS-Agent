## Changing the Order Channel

An order channel is where an order is placed. For example, guests can place an order at a kiosk, a drive thru window, inside a restaurant, or via a delivery aggregator. A default channel is assigned to checks, but can be changed if buttons have been configured on the POS to change the order channel.

Use the order channel buttons to change the order channel. If configured, the order channel appears in the check header.
