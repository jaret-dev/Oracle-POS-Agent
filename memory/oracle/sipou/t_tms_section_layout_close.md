# Closing a Section Layout

[Previous](t_tms_section_layout_activate_deactivate.htm) [Next](t_tms_host_server_availability.htm) JavaScript must be enabled to correctly display this content

## Closing a Section Layout

Follow these steps to close a layout and return to the active layout:

1.  Click the Close Layout button.
2.  If you made layout changes without saving, the application prompts you to save the changes or cancel without saving.
