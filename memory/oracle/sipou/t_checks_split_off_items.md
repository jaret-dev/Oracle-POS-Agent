# Splitting Off Items

[Previous](t_menu_items_using_nlu.htm) [Next](t_menu_items_split_suites_checks.htm) JavaScript must be enabled to correctly display this content

## Splitting Off Items

1.  To split off one item:
    1.  Select the menu item on the check, and then click Split Off Item.
    2.  Make the changes to the new check entry.
2.  To split off multiple items:
    1.  Select a multiple quantity menu item on the check.
    2.  Using the alphanumeric key pad, select the quantity that you want to split (for example, 2).
    3.  Click Split Off Item.
    4.  Make the changes to the new check entry.
