## 13 OPERA Guest Packages

Simphony allows users to query and view relevant information for a guest’s hotel reservation packages. This information is derived from the OPERA Cloud Rest API and displayed on the Simphony workstation. The displayed guest package information includes a description of the package, the total and remaining allowance for the guest, how the package is allocated across all guests staying in a room, and other basic pieces of information.
