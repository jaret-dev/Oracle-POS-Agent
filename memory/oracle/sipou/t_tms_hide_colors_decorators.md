# Hiding Colors and Table Decorators from a Table

[Previous](t_tms_host_server_availability.htm) [Next](t_tms_employee_section_assignment_report.htm) JavaScript must be enabled to correctly display this content

## Hiding Colors and Table Decorators from a Table

1.  Long-press on the Host Command area.
2.  Select one of the following options:
    
    -   To hide section colors, select Hide Sections.
        
    -   To hide employee colors, select Hide Employees.
        
    -   To hide decorators, select Hide Table Decorators.
