# Displaying OPERA Guest Packages Information

[Previous](c_opera_guest_packages.htm) [Next](c_opera_connect.htm) JavaScript must be enabled to correctly display this content

## Displaying OPERA Guest Packages Information

The following steps describe how a workstation operator can view and display OPERA Guest Package information.

1.  On the workstation, click the Display Opera Packages button.
2.  In the Enter Room Number and Last Name dialog window, enter the guest’s room number and last name in their corresponding fields.
3.  Click OK.
4.  A new dialog window opens, displaying the reservations for the entered room number. Click the desired reservation.
    
    Note:
    
    This window only appears if there is more than one guest assigned to the room number.
    
5.  Another dialog window opens, displaying the reservation package(s) and basic information about the package(s). To view more information for a package, click the desired package.
