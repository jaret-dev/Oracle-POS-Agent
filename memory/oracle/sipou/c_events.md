## 11 Events

An event is a group of transactions that occur within a specific date and time range in one or more revenue centers at a property. Examples of events include sporting events and games, banquets, weddings, and receptions. A property can track sales and inventory, particularly when multiple events occur in a single business day (for example, a baseball double header).
