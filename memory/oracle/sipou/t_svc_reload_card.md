# Reloading Gift Cards

[Previous](t_svc_activate_card.htm) [Next](t_svc_transfer_balance.htm) JavaScript must be enabled to correctly display this content

## Reloading Gift Cards

1.  Begin a check.
2.  Navigate to the gift and loyalty screen, and then select the function key to reload a gift card.
3.  Swipe the card or enter the card number.
4.  Enter the amount to add to the card.
5.  Pay the check to a tender, such as cash or credit card.
