## Seating Management

All seating requests on the Wait List automatically move up or down in the list depending on the configuration. Oracle Hospitality recommends that you seat requests at the top of the list first, working down the list.

After a guest table request appears on the Wait List (either by adding a wait request or a reservation that moved to the Wait List), and a table is available, hosting staff can seat the guest. You can use one of the following methods:

-   Seat the request from the Wait List on a host page
    
-   Perform a Quick Seat
