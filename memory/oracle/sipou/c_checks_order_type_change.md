## Changing the Order Type

An order type is a sales category based on where the order is consumed, such as Dine In or Take Out. Order types control active tax rates for a transaction, allocating a tax based on where the order is consumed. A default order type is assigned to checks, but can be changed if buttons have been configured on the POS to change the order type.

Use the order type buttons to change the order type. The order type appears in the check header.
