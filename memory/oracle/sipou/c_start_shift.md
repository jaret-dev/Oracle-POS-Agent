## 2 Start of Shift

Workstation operators sign on and sign off of the workstation securely. Workstation operators sign on by entering a unique employee ID number or by swiping an employee magnetic card. To sign on to a workstation, a workstation operator must have an employee ID number and be associated with an employee role that has the Allow Sign-In to a Workstation privilege.

If the employee is set up to clock in, the employee must first clock in before signing on to the workstation.
