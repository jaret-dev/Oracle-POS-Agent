# Transferring Gift Card Balance

[Previous](t_svc_reload_card.htm) [Next](t_svc_authorize_card.htm) JavaScript must be enabled to correctly display this content

## Transferring Gift Card Balance

1.  (Optional) Begin a check and add items.
2.  Navigate to the gift and loyalty screen, and then select the function key to transfer the gift card balance.
3.  Swipe the card or enter the card number of the account to transfer from.
4.  Swipe the card or enter the card number of the account to transfer to. Both funds and personal information (if available) transfer.
