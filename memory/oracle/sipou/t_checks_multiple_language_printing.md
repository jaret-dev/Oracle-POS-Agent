[Previous](t_checks_printer_selection.htm) [Next](t_email_receipt.htm) JavaScript must be enabled to correctly display this content

## Printing Guest Receipts in Multiple Languages

You can change the language for the customer receipt when more than two languages are configured.

1.  Sign on to the workstation.
2.  Add menu items.
3.  Click the Print Language List button.
4.  Select the language in the Change Print Language dialog box.
5.  Print the customer receipt.
6.  If the Print Language 2 button is configured, click Print Language 2.
