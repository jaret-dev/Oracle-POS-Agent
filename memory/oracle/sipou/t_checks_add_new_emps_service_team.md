# Adding Employees to an Existing Service Team

[Previous](t_checks_create_service_team.htm) [Next](t_checks_remove_emps_service_team.htm) JavaScript must be enabled to correctly display this content

## Adding Employees to an Existing Service Team

You need to have the appropriate privileges to perform this task.

1.  Press the Create Team Members or Add Team to Check function key.
2.  In the Team Service dialog, select the team to modify, and then press Edit.
3.  From the Select Employee section of the dialog, select the new employees to add to the team, and then press Add.
4.  Click Save.
