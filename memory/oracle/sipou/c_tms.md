## 10 Dining Room Management

Hosting staff can manage seating capacity, available server staff, guest seating preferences, dining table status, the wait list, and reservation list. When using the Table Management System (TMS), many functions require a long-press. To perform a long-press, right-click on the function or page area, hold down the right mouse click for a second, and then select an option.
