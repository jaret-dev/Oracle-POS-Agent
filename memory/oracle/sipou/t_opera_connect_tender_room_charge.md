# Tendering with OPERA Connection Room Charge

[Previous](t_opera_connect_guest_lookup.htm) [Next](c_messages.htm) JavaScript must be enabled to correctly display this content

## Tendering with OPERA Connection Room Charge

1.  Press the Room Charge tender.
2.  If guests are not assigned to the check:
    1.  Enter the guest's room number or last name.
    2.  Select the guest, and then press OK.
3.  If one guest is assigned to the check, you are prompted with Charge to \[Guest Name\]?. Select Yes to charge to the assigned guest or No to select a different guest.
4.  If multiple guests are assigned to the check, you are prompted with a list of all assigned guests or the option to select a new guest. Select the appropriate option.
5.  The tender is applied to the check and prints (if configured).
