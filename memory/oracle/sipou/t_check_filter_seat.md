# Filtering a Check by Seat Number

[Previous](c_check_filter_and_memo_check.htm) [Next](t_check_remove_filter.htm) JavaScript must be enabled to correctly display this content

## Filtering a Check by Seat Number

1.  Pick up a check that has menu items from multiple seats.
2.  To filter a single seat, use one of the following methods:
    1.  Press Add Seat to Filter, enter the seat number, and then press OK.
    2.  Press the seat number using the on-screen numeric key pad, and then press Add Seat to Filter.
3.  To filter multiple seats:
    1.  Press Manage Seat Filter.
    2.  From the Seat Filter window, select the seats in the Available Seats pane.
        
        Check details pertaining to the seats that you selected appear in the Selected Seat Check Detail pane.
        
    3.  Press Save
4.  To clear the seat filter, press Clear Seat Filter.
