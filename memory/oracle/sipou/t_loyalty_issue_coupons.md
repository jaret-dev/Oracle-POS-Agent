# Issuing Coupons

[Previous](t_loyalty_coupon_inquiry.htm) [Next](t_loyalty_redeem_coupon.htm) JavaScript must be enabled to correctly display this content

## Issuing Coupons

1.  (Optional) Begin a check and add items.
2.  Navigate to the gift and loyalty screen, and then select the function key to issue a coupon.
3.  Swipe the card or enter the loyalty account number. A list of ad hoc coupons appears.
4.  Select a coupon.
