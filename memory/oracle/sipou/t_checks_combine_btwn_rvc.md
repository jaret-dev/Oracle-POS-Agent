## Combining Checks Across Revenue Centers into One Order

There are two ways you can combine checks from different revenue centers: by check number and by table number.

1.  To view a list of all open checks in the revenue center before combining:
    1.  Pick up a check to combine with another.
    2.  Press the Add/Transfer from List Prompt RVC function key, select the revenue center that has the check you want to combine, and then press OK.
    3.  If the workstation has separate touchscreen buttons configured for each revenue center for viewing a list of all open checks, press the touchscreen button configured for combining checks from the revenue center that has the check. For example, if the check is in revenue center 1, press the Add/Transfer Check by List RVC 1 function key.
    4.  Select the check that you want to combine with the check that is currently open on the check detail area, and then press OK.
2.  To combine checks using the check number:
    1.  Pick up a check to combine with another.
    2.  Press the Add/Transfer Check by Number Prompt RVC function key, select the revenue center that has the check you want to combine, and then press OK.
    3.  If the workstation has separate touchscreen buttons configured for each revenue center for combining checks, press the touchscreen button configured for combining checks from the revenue center that has the check. For example, if the check is in revenue center 1, press the Add/Transfer Check by Number RVC 1 function key.
    4.  Enter the number of the check that you want to combine with the check that is currently open on the check detail area, and then press OK.
    5.  (Optional) Add more menu items to the check.
    6.  End the current service round.
3.  To combine checks using the table number:
    1.  Pick up a check to combine with another.
    2.  Press the Add/Transfer Check by Table Prompt RVC function key, select the revenue center that has the check you want to combine, and then press OK.
    3.  If the workstation has separate touchscreen buttons configured for each revenue center for combining checks, press the touchscreen button configured for combining checks from the revenue center that has the check. For example, if the check is in revenue center 1, press the Add/Transfer Check by Table RVC 1 function key.
    4.  Enter the table number that is on the check that you want to combine with the check that is currently open on the check detail area, and then press OK.
    5.  (Optional) Add more menu items to the check.
    6.  End the current service round.
