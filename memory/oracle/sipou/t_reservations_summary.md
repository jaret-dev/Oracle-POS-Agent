# Setting the Reservation Summary

[Previous](t_reservations_cancel.htm) [Next](r_reservations_icons.htm) JavaScript must be enabled to correctly display this content

## Setting the Reservation Summary

The summary area must be configured in order to view it in the Front of House.

1.  Double-tap quickly on the Reservation List to open it.
2.  Highlight a reservation request in the list. The summary area appears beneath the Reservation List and contains the following information about a reservation request.
    
    Table 10-7 Summary Area Options
    
    Summary Option
    
    Shows
    
    Cancel Date
    
    If the reservation request was cancelled, this summary option shows the date and time the request was cancelled or set to a no-show.
    
    Confirmation Number
    
    The confirmation number associated with the reservation request.
    
    Covers
    
    The number of covers associated with the reservation request.
    
    Creation Date
    
    The date and time the request was created.
    
    Greeted Date
    
    The time the request was greeted by a hosting staff member.
    
    Notes
    
    Text that has been included in the notes field of the reservation request.
    
    Phone Number
    
    The phone number associated with the reservation request.
    
    Referral Notes
    
    Text included in the referral notes field of the reservation request.
    
    Request Method
    
    A description of the request method for the reservation (for example, phone ahead, Internet).
    
    VIP
    
    A VIP flag if the guest has VIP status.
    
3.  To hide the summary area and prevent it from showing, long-press on a reservation request, and then select Hide Summary Area.
4.  To show the summary area, long-press on a reservation request, and then select Show Summary.
5.  To modify the summary area, long-press on a reservation request, and then select Modify Summary Area.
    1.  Select the appropriate summary options in the Select Items column.
    2.  To show the summary option on a separate line in the summary area, select the corresponding Add New Line check box. The text wraps when you deselect Add New Line.
    3.  Click OK.
