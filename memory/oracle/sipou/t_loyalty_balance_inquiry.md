# Performing a Loyalty Balance Inquiry

[Previous](c_loyalty_svc.htm) [Next](t_loyalty_issue_points.htm) JavaScript must be enabled to correctly display this content

## Performing a Loyalty Balance Inquiry

1.  (Optional) Begin a check and add items.
2.  Navigate to the gift and loyalty screen, and then select the function key for loyalty points inquiry.
3.  Swipe the card, enter the card number, or look up the account by name or phone number.
    
    If a guest’s card has been linked to multiple Stored Value and Gift card programs, select the program you wish to view.
    
4.  A message appears indicating the loyalty account balance.
5.  Click Yes to print the chit.
