# Applying a Coupon

[Previous](t_discounts_manual.htm) [Next](t_discounts_service_charge.htm) JavaScript must be enabled to correctly display this content

## Applying a Coupon

You can add a coupon to a check before or after adding the menu items. If the customer presents more than one coupon, Oracle recommends that you add the coupons after adding all menu items to the check. This allows the workstation to more easily add the discounts and calculate the check total.

1.  Select the button for the coupon.
2.  If the workstation prompts, enter the coupon reference details, and then click OK.
