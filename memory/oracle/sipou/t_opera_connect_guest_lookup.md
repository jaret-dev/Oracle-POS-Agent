## Guest Lookup

The OPERA Connection Guest Lookup function allows you to view details about a guest in a single view. Depending on the configuration, with OPERA Connection you can select one or more guests to add to a check or seat.

1.  Click the Guest Lookup button.
2.  Enter the room number or at least the first two letters of the guest’s last name. A list of matches appears.
    
    Each match is displayed as:
    
    `[Room Number] *`
    
    `[Last Name], [First Name] *`
    
    `Arrival: [date] – Departure: [date]`
    
    `Total Nights: [count]`
    
    \* If displayed in red, the guest is not allowed to charge to the room (No Post).
    
    1.  To view details about the guest, click the tile. The profile page includes the following information for the guest:
        
        `Room number and name`
        
        `Number of days until departure`
        
        `Number of adults/children in the room`
        
        `Preferences such as allergies`
        
        `Packages with details`
        
        `Package Descriptor`
        
        `Package Code`
        
        `Package Valid From — To`
        
        `Consumption Status`
        
        `Messages`
        
        `Notes`
        
        `Guest Notes`
        
        `Reservation Notes`
        
    2.  If a check is open, click the Assign Guest to Check or Assign Guest to Seat button to associate the guest with the check or seat.
        
        Assigning a guest does not pre-authorize nor charge payment; it is simply a query through OPERA Connection to associate the guest’s room number and name on the check. To tender the check using OPERA Connection Room Charge, see [Tendering with OPERA Connect Room Charge](t_opera_connect_tender_room_charge.htm).
        
        If seat handling is enabled in the revenue center, you can assign a guest to a specific seat for more personalized service.
