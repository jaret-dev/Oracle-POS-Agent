# Declaring Waste

[Previous](t_menu_items_popup_condiment_orderer.htm) [Next](t_menu_items_correct_waste_check.htm) JavaScript must be enabled to correctly display this content

## Declaring Waste

1.  Click the Declare Waste function key.
2.  Enter the menu item. If more than one of the same item is wasted, select the quantity, and then select the menu item to add to the waste check.
3.  Select a waste reason, and then click OK.
4.  Repeat Steps 2 and 3 to add menu items to the waste check.
5.  Perform a service total on the check. A waste receipt prints and the workstation closes the waste check.
