# Custom Browser

## 6 Custom Browser

With this feature, you add buttons on touchscreen pages which launch web pages on the workstation. Press buttons for each URL or a button, when pressed, shows a selection dialog with a list of the available URLs for selection.

A Close button enables you to close the webpage.

The following error is shown when you attempt to access a webpage which has not been properly configured: URL is blocked. Your organization doesn’t allow you to view this site.

For your information, this is a Microsoft Edge generated error message and cannot be customized.

Key Points

-   This feature is only supported on workstations running Microsoft Windows which have a minimum version of Microsoft Edge 79.0.309.0.
    
-   A new Browser User has been added to the workstation for this feature.
    
-   Downloads are not allowed when using this feature.
    
-   Policies on the local machine take precedence over those of the new Browser User level.
    
-   This feature requires CAL 160 and above to work.
    
-   This functionality uses Page Visibility to restrict who can access the web page links.
