# Activating and Deactivating a Section Layout

[Previous](t_tms_section_layout_edit.htm) [Next](t_tms_section_layout_close.htm) JavaScript must be enabled to correctly display this content

## Activating and Deactivating a Section Layout

Activate a layout to assign employees to sections and tables. Activating a new layout establishes the next employee to open a guest check on a table. Changing the active layout does not change the ownership of open guest checks.

Deactivate a layout to quickly clear all section and employee assignments for the tables. Tables with open guest checks appear with the Occupied status. Guest check ownership does not change when deactivating a layout.

1.  To activate a section layout:
    1.  Click the Load Layout button.
    2.  Select the layout to apply, and then click the Activate button. The layout appears as the active table layout.
2.  To deactivate a section layout:
    1.  Click the More... button.
    2.  Select Deactivate Layout. Assigned sections and tables disappear.
