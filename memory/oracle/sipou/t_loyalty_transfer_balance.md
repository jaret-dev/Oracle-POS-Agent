# Transferring Points Balance

[Previous](t_loyalty_redeem_coupon.htm) [Next](t_loyalty_unique_items_inquiry.htm) JavaScript must be enabled to correctly display this content

## Transferring Points Balance

1.  (Optional) Begin a check and add items.
2.  Navigate to the gift and loyalty screen, and then select the function key to transfer points balance.
3.  Swipe the card or enter the card number of the account to transfer from.
4.  Swipe the card or enter the card number of the account to transfer to. Both points and personal information transfer.
