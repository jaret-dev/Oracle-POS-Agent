## Check Transfers

Transferring a check allows a workstation operator to take ownership of a check from another workstation operator or another revenue center. For example, a couple arrives at a restaurant to find that there are no tables available, so they go to the bar for drinks. The bartender starts the check. When a table opens, the check is transferred from the bartender to a server in the dining room, so the customer only has to pay one check at the end of the visit.

Workstation operators can transfer checks between revenue centers only when the automatic service charge for the revenue centers are the same. The following table provides examples and explains when you can transfer checks between revenue centers:

To take ownership of a check you must be on the default transaction screen. If you attempt to transfer while in a transaction (a check is open on the check detail area), the workstation merges the checks rather than transferring ownership.
