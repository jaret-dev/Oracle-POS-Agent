# Changing a Section Layout

[Previous](t_tms_section_layout_create_delete.htm) [Next](t_tms_section_layout_activate_deactivate.htm) JavaScript must be enabled to correctly display this content

## Changing a Section Layout

1.  Click the Load Layout button.
2.  Select the layout you want to change, and then click the Load button.
    
    If you selected a section that was created from a template, table numbers could be assigned to sections within the list. You can alter the table numbers for the layout. If the section was not created from a template, no assignments exist for the section.
    
3.  To assign a table to a section:
    1.  Highlight the section.
    2.  Touch the table on the workstation screen to add the table number to the section.
    3.  Long-press on the section, and then select Assign Table.
    4.  Select the appropriate Table ID and Table Name to assign. The POS client shows a list of tables that are not associated with a section.
        
        You can select Unassign Table to remove a Table ID and Table Name from the section.
        
4.  To assign an employee to a section:
    1.  Long-press on the section, and then select Assign Employee.
    2.  Select the employee to assign. If the section has an employee assigned, None appears to allow you to unassign the employee from the section.
5.  Click the Save button.
    
    The Save As button allows you to save a copy of the loaded layout with a different layout name. You can use the Save As button to make minor changes from one layout to another rather than re-creating a new layout.
    
    1.  Click the More... button.
    2.  Select Save As.
    3.  Enter a name that reflects the purpose of the layout you are saving, and then click OK.
