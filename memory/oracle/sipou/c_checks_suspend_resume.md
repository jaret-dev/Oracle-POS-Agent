## Suspend and Resume Checks

With Suspend/Resume you can:

-   Suspend a check started by another employee, and sign on to the workstation.
    
-   Suspend a check when the workstation automatically signs out an employee.
    
-   Suspend a check using the Suspend Order Tender/Media key.
    

When suspended, the check is stored in the memory until it is resumed.
