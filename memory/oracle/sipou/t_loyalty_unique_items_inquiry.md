[Previous](t_loyalty_transfer_balance.htm) [Next](t_loyalty_apply_card_check.htm) JavaScript must be enabled to correctly display this content

## Performing Unique Items Inquiry

You can perform this operation only with Oracle Hospitality Gift and Loyalty Advanced Cloud Service. You cannot perform this operation with a third-party loyalty application.

1.  (Optional) Begin a check and add items.
2.  Navigate to the gift and loyalty screen, and then select the function key for unique items inquiry.
3.  Swipe the card or enter the loyalty account number. A list of menu items that were ordered for the loyalty program appears.
4.  (Optional) Print the list of items.
