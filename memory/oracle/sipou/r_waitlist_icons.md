# Wait List Icons

[Previous](t_waitlist_summary.htm) [Next](r_tms_employee_section_assignment_report.htm) JavaScript must be enabled to correctly display this content

## Wait List Icons

The following icons appear on the Wait List.

Table 10-12 Wait List Icons

Icon

Icon Name

Description

![This figure shows the Call Guest icon, which is a green circle with a smiley face.](img/CallGuestIcon.png "This figure shows the Call Guest icon, which is a green circle with a smiley face.")

Call Guest

Hosting staff paged the guest.

![This figure shows the Phone Reservation icon, which is a green circle with a phone.](img/PhoneReservationIcon.png "This figure shows the Phone Reservation icon, which is a green circle with a phone.")

Phone Reservation

The guest phoned to make the reservation, and hosting staff greeted the guest.

![This figure shows the Phone Reservation (Ungreeted) icon, which is an orange circle with a phone.](img/PhoneReservationUngreetedIcon.png "This figure shows the Phone Reservation (Ungreeted) icon, which is an orange circle with a phone.")

Phone Reservation (Ungreeted)

The guest phoned to make the reservation, but hosting staff has not greeted the guest.

![This figure shows the Internet Reservation icon, which is a green circle with the at symbol.](img/InternetReservationIcon.png "This figure shows the Internet Reservation icon, which is a green circle with the at symbol.")

Internet Reservation

The guest made the reservation online, and hosting staff greeted the guest.

![This figure shows the Internet Reservation (Ungreeted) icon, which is an orange circle with the at symbol.](img/InternetReservationUngreetedIcon.png "This figure shows the Internet Reservation (Ungreeted) icon, which is an orange circle with the at symbol.")

Internet Reservation (Ungreeted)

The guest made the reservation online, but hosting staff has not greeted the guest.

![This figure shows the VIP icon, which is a blue circle with a star.](img/VIPIcon.png "This figure shows the VIP icon, which is a blue circle with a star.")

VIP

The reservation is for a VIP guest.

![This figure shows the Reservations icon, which is a light blue circle with a clock.](img/ReservationsIcon.png "This figure shows the Reservations icon, which is a light blue circle with a clock.")

Reservations

The reservation was generated from the Reservation List (not from the Wait List).

![This figure shows the Wait Quote Threshold Exceeded icon, which is an orange circle with an exclamation point.](img/WaitQuoteThresholdExceededIcon.png "This figure shows the Wait Quote Threshold Exceeded icon, which is an orange circle with an exclamation point.")

Wait Quote Threshold Exceeded

The amount of time the hosting staff quoted to the guest has elapsed. The request is overdue for seating.

![This figure shows the Table Ready icon, which is a gray circle with a table and chairs.](img/TableReadyIcon.png "This figure shows the Table Ready icon, which is a gray circle with a table and chairs.")

Table Ready

The table is now ready to seat the request.

![This figure shows the Table Preference icon, which is a blue circle with a tiny square, circle, and triangle.](img/TablePreferenceIcon.png "This figure shows the Table Preference icon, which is a blue circle with a tiny square, circle, and triangle.")

Table Preference

The wait or reservation record contains a request for a specific type of table.
