## Engagement Auditing Tool

You can track a variety of operations using the Engagement configurator. The auditing tool tracks and provides the date and time, user ID, user name, actions that were performed, and additional details about each operation.

The Engagement auditing tool provides information about the operations listed in the following table:

Table 9-1 Engagement Auditing Tool Tracking Information

Operation Category

Operation

Administrative (User)

-   Successful and failed logins
    
-   Log outs from the configurator
    
-   Synchronizations performed
    

Page

-   Creating, deleting, and cloning a page
    
-   Changing the theme of a page
    
-   Assigning, removing, and cropping a logo on a page
    

Widget

-   Adding and removing a widget on a page
    
-   Saving a widget’s template name
    
-   Changing the size of a widget (both previous and new sizes)
    
-   Changing the widget name (both previous and new names)
    
-   Uploading, removing, and cropping a widget background image
