## Unseating a Table

If the guest wants to move to another table after being seated, you can unseat them and re-seat the guest at a different table. You can only unseat a table if the guest has not ordered menu items. After a guest check is open on a table, you cannot use the Unseat Table feature.

Perform a long-press on the table with the seated guest, and then select Unseat Table.

-   If the guest was previously seated from the Wait List, the guest’s name appears on the Wait List.
    
-   If the guest was previously seated using Quick Seat, the guest appears on the Wait List as QS in the guest name column.
