## Voiding a Gift Card

If you are using SPI and you do not have privileges to void service charges from a previous round, when you attempt to void a gift card or a menu item that was added before the gift card (in the current round), authorization is required.

1.  Create a check with the gift card.
2.  Select the gift card.
3.  Press the Void function key.
