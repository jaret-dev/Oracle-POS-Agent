# Issuing Loyalty Points

[Previous](t_loyalty_balance_inquiry.htm) [Next](t_loyalty_coupon_inquiry.htm) JavaScript must be enabled to correctly display this content

## Issuing Loyalty Points

1.  Begin a check and add items that trigger loyalty points.
2.  Navigate to the gift and loyalty screen, and then select the function key to issue points to a loyalty account.
3.  Swipe the card or enter the loyalty account number. A message appears indicating the points issuance.
4.  Perform either a service total or tender the check.
