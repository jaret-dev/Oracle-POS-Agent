# Pay Using QR Code

[Previous](t_payment_quick_split_equal_payments.htm) [Next](t_payment_consumer_self_pay.htm) JavaScript must be enabled to correctly display this content

## Pay Using QR Code

Click the Service Total function key. The QR code appears on the printed guest check.

The guest scans the QR code on their check, which opens a browser on their mobile device and loads the check. After payment is complete, the check is closed in Simphony and shows the payment method that was used by the guest.
