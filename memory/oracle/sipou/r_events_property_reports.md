## Event Workstation Reports

The workstation property reports contain information about events. The following table lists the reports that contain event-related information.

Table 11-1 Event Property Reports

Report

Description

Property Financial Event Report

Shows events for each property for the period that you select, including total event and non-event sales, and outstanding (open) checks associated with events.

Property Financial Event VAT Report

Shows Property Financial Event information, with the addition of value-added tax (VAT) detail.

Employee Financial Event Report

Shows events for the period and employee that you select, including total event and non-event sales.

Employee Financial Event VAT Report

Shows Employee Financial Event information, with the addition of value-added tax (VAT) detail.

Event Financial Report

Shows the event and period that you select, including total event and non-event sales.

Event Financial VAT Report

Shows Event Financial information, with the addition of value-added tax (VAT) detail.

Menu Item Sales Event Report

Shows menu items ordered during the period and event you select, including total event and non-event sales.

Employee Journal Report

Provides a journal of all sales transactions by a specific employee shown in a check detail layout. The information appears by revenue center and then by event.

Check Journal Report

Provides a journal of all sales transactions in a check detail layout. The information appears by revenue center and then by event.

Employee Open Check Report

Provides a list of all open checks belonging to an employee at the time the report is taken. Checks are grouped in the following order: employee, property or revenue center, and event. Event details appear in the check information.

Employee Closed Check Report

Provides a list of all checks that have been closed by an employee, including reopened checks that were closed again and checks closed as memo checks. In addition, special symbols on the report indicate whether the check was transferred, re-opened, split, or added. Checks are grouped in the following order: employee, property or revenue center, and event. Event details appear in the check information.

Offline Employee Open Check Report

Provides a list of all open checks belonging to an employee at the time the report is taken. Checks are grouped in the following order: employee, revenue center, and event. Event details appear in the check information.

Unlike the online version of this report, the offline version functions only for a revenue center; it is not possible to take a report for the property.

Employee Autofire Open Check Report

Provides a list of all checks that are scheduled for autofire at the time the report is run. Checks are grouped by employee, property or revenue center, and event, and sorted by scheduled date and time. Event details appear in the check information.
