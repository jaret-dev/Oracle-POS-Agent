# Adding a Service Team to an Open Check

[Previous](t_checks_create_team_check.htm) [Next](t_checks_remove_service_team_from_check.htm) JavaScript must be enabled to correctly display this content

## Adding a Service Team to an Open Check

You need to have the appropriate privileges to perform this task.

1.  Pick up the appropriate check, and then press the Create Team Members or Add Team to Check function key.
2.  Select the service team to add to the check, and then press Select.
3.  To create a new service team, press Add.
