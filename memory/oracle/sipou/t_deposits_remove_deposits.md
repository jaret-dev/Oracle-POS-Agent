# Removing a Deposit From a Check

[Previous](t_deposits_modify.htm) [Next](t_deposits_view_and_search.htm) JavaScript must be enabled to correctly display this content

## Removing a Deposit From a Check

1.  Pick up the check with the deposit.
2.  Select the deposit information on the check, and then press the Void function key.
3.  Select the rental menu item or service charge, and then press the Void function key.
4.  Perform either a service total or tender the check.
