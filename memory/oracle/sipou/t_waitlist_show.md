# Showing and Hiding the Wait List

[Previous](c_waitlist.htm) [Next](t_waitlist_add.htm) JavaScript must be enabled to correctly display this content

## Showing and Hiding the Wait List

1.  To view the wait list, long-press on the Host Command area, and then select Show Wait List.
2.  To hide the wait list, long-press on the Host Command area, and then select Hide Wait List.
