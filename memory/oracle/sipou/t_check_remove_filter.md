# Removing a Seat from the Filter

[Previous](t_check_filter_seat.htm) [Next](t_check_edit_seat.htm) JavaScript must be enabled to correctly display this content

## Removing a Seat from the Filter

When a check is filtered by one or more seats, you can remove a seat from the filtered list at any time. To remove a seat from the filter:

1.  Press the Remove From Seat Filter function key.
2.  Enter the seat number to remove, and then press OK.
