# Applying a Return

[Previous](t_discounts_service_charge.htm) [Next](t_discounts_svc_chrg_condiment.htm) JavaScript must be enabled to correctly display this content

## Applying a Return

1.  To return one menu item:
    1.  Pick up a check or begin a new check, and then click the Return function key.
    2.  Add the menu item to return to the check.
    3.  End the current service round or close the check.
2.  To return more than one menu item:
    1.  Pick up a check or begin a new check, and then click the Transaction Return function key.
    2.  Add the menu items to return to the check.
    3.  End the current service round or close the check.
