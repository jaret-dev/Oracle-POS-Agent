## 3 Guest Checks

A guest check is a record of all sales on a check, from the customer’s first order to final payment. Guest check detail includes the following information:

-   Menu Items
    
-   Discounts
    
-   Service Charges
    
-   Voids
    
-   Returns
    
-   Tenders
