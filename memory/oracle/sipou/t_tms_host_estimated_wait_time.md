# Viewing Estimated Wait Times

[Previous](c_waitlist_seating.htm) [Next](t_reservations_waitlist_greet.htm) JavaScript must be enabled to correctly display this content

## Viewing Estimated Wait Times

The Estimated Wait Times option shows the estimated amount of time remaining until the application anticipates each request being seated.

1.  Long-press on the Host Area, and then select Estimated Wait Times. The following information appears in the columns:
    
    -   Cover Count: The number of people that the table capacity allows.
        
    -   Estimated Wait Time: The estimated time remaining until a table is available for the cover count.
        
    -   Projected Table: The system suggested table number for the request.
        
    -   Projected Server: When employees are assigned to tables, the probable server name appears.
        
    
2.  Click OK to close the Estimated Wait Times list.
