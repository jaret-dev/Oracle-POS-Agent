## Viewing the Employee Section Assignment Report

Follow these steps to run the Employee Section Assignment Report:

1.  Open the Reports List on the workstation.
2.  Select the Employee Section Assignment Report, or enter the Report Number, and then click OK.
3.  (Optional) Change the revenue center in the View field.
4.  Click the Run Report button. Results appear in the Print, Display, and Stats tabs of the Reports form.
    
    If you do not have privileges to run the report, a message appears and the application returns to the Reports list.
    
5.  (Optional) To print the report on the receipt printer, click the Print button.
6.  Click the Close button to close the Reports form.
