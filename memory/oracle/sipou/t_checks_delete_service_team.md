# Deleting a Service Team

[Previous](t_checks_print_service_team.htm) [Next](t_checks_create_team_check.htm) JavaScript must be enabled to correctly display this content

## Deleting a Service Team

You need to have the appropriate privileges to perform this task.

1.  Press the Create Team Members or Add Team to Check function key.
2.  In the Team Service dialog, select the team to delete.
3.  To delete all teams, select Select All.
4.  Press Delete.
