# Applying a Condiment Discount

[Previous](t_discounts_return.htm) [Next](c_discounts_bundle.htm) JavaScript must be enabled to correctly display this content

## Applying a Condiment Discount

You can apply a condiment discount manually, automatically, or with a coupon.

1.  To apply a discount manually:
    1.  Add a parent menu item to the guest check.
    2.  Add the condiment that is configured to apply a discount.
    3.  Click the button that is configured for the manual condiment trigger function. Simphony applies the discount to the guest check.
    4.  Close the transaction.
2.  To apply a discount automatically:
    1.  Add a parent menu item to the guest check.
    2.  Add the condiment that is configured to apply a discount. Simphony applies the discount to the guest check.
    3.  Close the transaction.
3.  To apply a discount using a coupon:
    1.  Add a parent menu item to the guest check.
    2.  Add the condiment that is configured to apply a discount.
    3.  Click the button that is configured for the coupon condiment trigger function. Simphony applies the discount to the guest check.
    4.  Close the transaction.
