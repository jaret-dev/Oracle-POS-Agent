[Previous](c_tms_host_employee_lines.htm) [Next](c_tms_table_status.htm) JavaScript must be enabled to correctly display this content

## Section Layout

The Seating Section Layout allows hosting staff to quickly select tables to be used in the revenue center, and assign section and employees. You can manage the Section Layout on its own page or as a popup in the host view.

To show the Section Layout, long-press on the Host Command area and select Show Section Layout.

To hide the Section Layout, long-press on the Host Command area and select Hide Section Layout.
