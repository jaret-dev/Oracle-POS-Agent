# Transferring a Check Within a Revenue Center

[Previous](c_checks_transfers.htm) [Next](t_checks_transfer_btwn_rvc.htm) JavaScript must be enabled to correctly display this content

## Transferring a Check Within a Revenue Center

There are three ways you can transfer open checks from one server to another within the same revenue center: simultaneously, by check number, and by table number.

If the revenue center uses events, you can transfer open checks from one server to another within the same event or event area. You cannot transfer open checks across different events or areas. When transferring checks, both checks must have the same order type or order channel assigned.

1.  To view a list of all open checks in the revenue center before transferring:
    1.  Press the Add/Transfer Check SLU function key.
    2.  Select a check from the list, and then press OK.
2.  To transfer all checks belonging to another workstation operator simultaneously:
    1.  Press the Auto Block Check Transfer function key.
    2.  Select an employee or enter the ID of the employee whose checks you want to transfer, and then press OK twice to confirm.
3.  To transfer a check using the check number:
    1.  Press the Add/Transfer Check by Number function key.
    2.  Enter the check number, and then press OK twice to confirm.
4.  To transfer a check using the table number:
    1.  Press the Add/Transfer Check by Table function key.
    2.  Enter the table number, and then press OK twice to confirm.
