## Transferring a Check to Another Revenue Center

There are two ways you can transfer a check to another revenue center: by check number and by table number. When transferring checks, both checks must have the same order type or order channel assigned.

1.  To view a list of all open checks in the revenue center before transferring:
    1.  Press the Add/Transfer from List Prompt RVC function key, select the revenue center that has the check you want to transfer, and then press OK.
    2.  If the workstation has separate touchscreen buttons configured for each revenue center for viewing a list of all transferable checks, press the touchscreen button configured for transferring checks from the revenue center that has the check. For example, if the check is in revenue center 1, press the Add/Transfer Check by List RVC 1 function key.
    3.  Select the check that you want to transfer from the list, and then press OK.
2.  To transfer a check using the check number:
    1.  Press the Add/Transfer Check by Number Prompt RVC function key, select the revenue center that has the check you want to transfer, and then press OK.
    2.  If the workstation has separate touchscreen buttons configured for each revenue center for transferring checks, press the touchscreen button configured for transferring checks from the revenue center that has the check. For example, if the check is in revenue center 1, press the Add/Transfer Check by Number RVC 1 function key.
    3.  Enter the check number, and then press OK twice to confirm.
3.  To transfer a check using the table number:
    1.  Press the Add/Transfer Check by Table Prompt RVC function key, select the revenue center that has the check you want to transfer, and then press OK.
    2.  If the workstation has separate touchscreen buttons configured for each revenue center for transferring checks, press the touchscreen button configured for transferring checks from the revenue center that has the check. For example, if the check is in revenue center 1, press the Add/Transfer Check by Table RVC 1 function key.
    3.  Enter the table number, and then press OK twice to confirm.
