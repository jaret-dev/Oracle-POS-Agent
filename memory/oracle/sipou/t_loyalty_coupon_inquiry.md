# Performing a Coupon Inquiry

[Previous](t_loyalty_issue_points.htm) [Next](t_loyalty_issue_coupons.htm) JavaScript must be enabled to correctly display this content

## Performing a Coupon Inquiry

1.  (Optional) Begin a check and add items.
2.  Navigate to the gift and loyalty screen, and then select the function key to look up coupons.
3.  Swipe the card or enter the loyalty account number. A list of available coupons appears.
    
    -   If a check is open, select a coupon to redeem.
        
    -   If a check is not open, click Yes to print the coupon.
