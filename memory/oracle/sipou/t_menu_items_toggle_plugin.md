## Menu Item Information Toggle Function

If the Menu Item Information extension application plug-in is configured, workstation operators can use the toggle function on the workstation to view menu item information.

1.  If items are not selected in the check detail for an open check, click Menu Item Information, and then click Ok to enable the display of the next menu item you click.
    
    Figure 4-5 Menu Item Information Enabled Message
    
      
    ![This dialog says Menu item information enabled. The next menu item button pressed will display the Allergen & Nutrition data for that item.](img/ext_app_menu1.png "This dialog says Menu item information enabled. The next menu item button pressed will display the Allergen & Nutrition data for that item.")  
    
    Simphony returns to regular operations after displaying the menu item information.
    
2.  To disable the dialog, ensure that menu items are not selected, click Menu Item Information, and then click Ok.
    
    Figure 4-6 Menu Item Information Disabled
    
      
    ![This dialog says Menu Item Information Disabled.](img/ext_app_menu2.png "This dialog says Menu Item Information Disabled.")  
    
3.  With the Menu Item Information display enabled, click a menu item. The menu item information is displayed.
    
    Figure 4-7 Example of Menu Item Information
    
      
    ![This image shows the menu item information on the workstation.](img/ext_app_menu_allergen.png "This image shows the menu item information on the workstation.")  
    
    1.  To add this menu item to the check, click Add.
    2.  To print a copy of the menu item information on the receipt printer, click Print.
    3.  To close the menu item information page, click Back.
