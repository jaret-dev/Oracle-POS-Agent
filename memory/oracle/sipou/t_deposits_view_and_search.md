## Viewing and Searching for Deposits

1.  Press the Show Deposit function key to open the Select Deposit dialog.
2.  To view a list of all deposits made, press Search.
3.  Use the following filters to narrow the results, and then press Search:
    
    -   Deposit ID: Enter the deposit number in the receipt.
        
    -   First Name: Enter the customer’s first name.
        
    -   Last Name: Enter the customer’s last name.
        
    -   Check #: Enter the check number.
        
    -   Phone #: Enter the customer’s phone number.
        
    -   Description: Enter any key words specified when creating the deposit.
        
    -   Custom Start: Enter the start date to search for a deposit by a date range.
        
    -   Custom End: Enter the end date to search for a deposit by a date range.
