# Redeeming Gift Cards

[Previous](t_svc_authorize_card.htm) [Next](t_svc_cash_out.htm) JavaScript must be enabled to correctly display this content

## Redeeming Gift Cards

1.  Begin a check.
2.  Navigate to the gift and loyalty screen, and then select the function key to redeem a gift card.
3.  Swipe the card or enter the card number.
    
    If a guest’s card has been linked to multiple Stored Value and Gift card programs, select the program to redeem.
    
4.  Enter the amount to redeem from the card.
    
    -   If the amount is less than or equal to the gift card balance, the card balance is applied to the check and deducted from the card.
        
    -   If the amount exceeds the gift card balance, the full card balance is applied to the check and the card balance becomes 0.00. Apply another tender (cash, credit card, or another gift card) to pay the remaining check balance.
        
    -   If the check amount is zero, enter a value of zero when redeeming the card.
        
    
    Note:
    
    Some gift card drivers (using the Command Module / XProcessor) use a separate Gift Card Redemption function button. For Simphony Payment Interface (SPI) users, gift cards are redeemed by using the SPI payment function buttons (CC Auth, CC Final, or Authorization & Pay).
