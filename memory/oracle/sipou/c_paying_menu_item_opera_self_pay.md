## OPERA Self Pay Charges to a Room

OPERA Self Pay enables a guest to pay the check by adding payment to a room folio. OPERA Self Pay is supported with Simphony release 19.1 or later and the Oracle Hospitality OPERA 5 Hotel Property Systems release 5.0.04.01 or later.

The workstation operator adds menu items in the POS client using one of the following handheld devices:

-   Oracle MICROS Tablet E-Series
    
-   Tablet devices running the Android mobile operating system
    
-   Zebra MC40 (formerly the Motorola MC40) mobile devices running on the Android mobile operating system and configured with either an integrated or external magnetic card reader
    

At the end of the dining experience or when delivering the order to a guest in a hotel room, the workstation operator presents a tablet device to the guest with the check. The guest can:

-   View the check
    
-   Add a tip or change the tip value
    
-   Select a different language
    
-   Charge entire check to a room account
    
-   Enter a room number and guest last name
    
-   Sign the check with electronic signature after authorization
    
-   Enter a valid email address and receive a receipt electronically through email
